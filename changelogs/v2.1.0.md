# ZyroxCSS v2.1.0 - JavaScript Components Release

**Release Date**: March 6, 2026  
**Status**: Stable & Production Ready  
**Type**: Minor Release

---

## 🎉 Overview

v2.1.0 adds comprehensive JavaScript functionality to ZyroxCSS, bringing interactive components to life. All previously static CSS components now have full JavaScript support for enhanced user experience.

---

## ✨ New Features

### JavaScript Components Library
- **Modal System**: Complete modal functionality with open/close triggers, escape key support, and backdrop clicks
- **Dropdown Menus**: Interactive dropdowns with toggle functionality and mobile support
- **Carousel Component**: Image carousel with navigation controls and optional auto-play
- **Tab System**: Dynamic tab switching with content panels
- **Toast Notifications**: Programmatic toast messages with auto-dismiss and manual close
- **Enhanced Tooltips**: Touch device support for tooltips
- **Alert Dismissal**: Click-to-dismiss functionality for alerts

### API Methods
```javascript
// Initialize all components
const zyrox = new ZyroxCSS();

// Manual modal control
zyrox.openModal('#my-modal');
zyrox.closeModal(document.querySelector('#my-modal'));

// Show toast notifications
zyrox.showToast('Success message!', 'success', 5000);
zyrox.showToast('Error occurred', 'error');
```

### Data Attributes
- `data-modal-target`: Opens specified modal
- `data-modal-close`: Closes modal
- `data-autoplay`: Enables carousel auto-play (milliseconds)
